podpac.algorithm¶

podpac.algorithm.Algorithm(**kwargs) Base node for any algorithm or computation node.
podpac.algorithm.Arithmetic(**kwargs) Create a simple point-by-point computation of up to 7 different input nodes.
podpac.algorithm.SinCoords(**kwargs) A simple test node that creates a data based on coordinates and trigonometric (sin) functions.
podpac.algorithm.Arange(**kwargs) A simple test node that gives each value in the output a number.
podpac.algorithm.CoordData(**kwargs) Extracts the coordinates from a request and makes it available as a data

stats

podpac.algorithm.Min(**kwargs) Computes the minimum across dimension(s)
podpac.algorithm.Max(**kwargs) Computes the maximum across dimension(s)
podpac.algorithm.Sum(**kwargs) Computes the sum across dimension(s)
podpac.algorithm.Count(**kwargs) Counts the finite values across dimension(s)
podpac.algorithm.Mean(**kwargs) Computes the mean across dimension(s)
podpac.algorithm.Median(**kwargs) Computes the median across dimension(s)
podpac.algorithm.Variance(**kwargs) Computes the variance across dimension(s)
podpac.algorithm.StandardDeviation(**kwargs) Computes the standard deviation across dimension(s)
podpac.algorithm.Skew(**kwargs) Computes the skew across dimension(s)
podpac.algorithm.Kurtosis(**kwargs) Computes the kurtosis across dimension(s) TODO NaN behavior when there is NO data (currently different in reduce and reduce_chunked)
podpac.algorithm.DayOfYear(**kwargs) Group a time-dependent source node by day of year and reduce.
podpac.algorithm.GroupReduce(**kwargs) Group a time-dependent source node by a datetime accessor and reduce.
podpac.algorithm.Reduce(**kwargs) Base node for reduction algorithms
podpac.algorithm.Reduce2(**kwargs) Extended Reduce class that enables chunks that are smaller than the reduced output array.

coordinates

podpac.algorithm.ExpandCoordinates(**kwargs) Algorithm node used to expand requested coordinates.
podpac.algorithm.SelectCoordinates(**kwargs) Algorithm node used to select coordinates different from the input coordinates.

signal

podpac.algorithm.Convolution(**kwargs) Base algorithm node for computing convolutions.
podpac.algorithm.SpatialConvolution(**kwargs) Specialized convolution node that computes lat-lon convolutions only.
podpac.algorithm.TimeConvolution(**kwargs) Specialized convolution node that computes temporal convolutions only.
