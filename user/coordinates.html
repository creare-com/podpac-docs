Coordinates¶

Coordinates are used to:

  1. Execute nodes which retrieve and process data
  2. Define the coordinates of data sources

The interface for Coordinates are currently unstable. To peek at what we’re thinking, see Developer Coordinates Spec

For now, see the Notebook Examples for examples of creating Coordinates.
