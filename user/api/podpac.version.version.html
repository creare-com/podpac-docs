podpac.version.version¶

podpac.version.version()[source]¶

Retrieve PODPAC semantic version as string

Returns:Semantic version if on the master branch (or outside git repository). Includes +git hash if in a git repository off the master branch (i.e. 0.0.0+hash)
Return type:str
